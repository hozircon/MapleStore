# MapleStore

## 開發環境設置

### 遊戲資源檔案（`resource/`）

`resource/` 目錄因體積過大，**不納入 git 版本控制**。

請從共享儲存空間取得後，放到專案根目錄的 `resource/` 下，結構如下：

```
resource/
├── Character_0_img.wz/
├── Character_0_xml.wz/
├── Item.wz/
└── String.wz/
```
